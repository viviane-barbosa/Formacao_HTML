Tag h1 a h6

As tags h1 até a h6 são tags para títulos em uma página, onde o h1 irá representar um título maior e o h6 a menor versão do título, tendo as variações no intervalo do h2, h3, h4 e h5.

Tag p

A tag p deve ser utilizada para inserir paragrafos em página html.

Tag i

A tag i deixa o texto em itálico.

Tag u

A tag u deixa o texto como sublinhado.

Tag ol e ul

As tags ol e lu são tags de lista. Tag ol é de lista ordenada, onde ao ser utilizada irá criar uma listar ordenada de itens (1, 2, 3...), já a tag ul é de lista não ordenada, onde ao ser utilizada irá criar uma lista de itens de forma não ordenada.

Tag mark

A tag mark insere um marcador de texto na página.
